# Agent Task Page Template

아래 템플릿을 개발과제 페이지에 그대로 적용한다.

## 1. 페이지 제목

```text
[구분] 과제명
```

예:

```text
[채용] JD 초안 생성 에이전트
```

## 2. 페이지 label

예:

```text
agent-task
bu-vd
cat-recruiting
```

## 3. Content Properties 안에 넣을 표

`Content Properties` 매크로를 먼저 삽입한 다음, 그 안에 아래 2열 표를 만든다.

| 항목 | 값 |
| --- | --- |
| 사업부 | VD |
| 구분 | 채용 |
| 과제명 | JD 초안 생성 에이전트 |
| 목적 | 채용 담당자의 JD 작성 시간을 줄이고 초안 품질을 높인다 |
| 담당 | 홍길동 |
| 상태 | 진행중 |
| 우선순위 | High |
| 완료예정일 | 2026-04-30 |
| 관련 링크 | Jira-123 / 문서 링크 / 데모 링크 |

## 4. 본문 권장 섹션

| 섹션 | 내용 |
| --- | --- |
| 개요 | 왜 필요한 과제인지 |
| 현재 문제 | 현재 업무의 병목 |
| 기대 효과 | 시간 절감, 품질 향상 등 |
| 범위 | 이번 단계에 포함 / 제외할 것 |
| 참고 자료 | 정책 문서, 업무 흐름, 샘플 파일 |
| 진행 메모 | 회의 메모, 이슈, 결정 사항 |

## 5. 복붙용 예시 본문

```markdown
## 개요
채용 담당자가 JD 초안을 만드는 데 걸리는 시간을 줄이기 위한 에이전트 과제다.

## 현재 문제
- 유사 JD를 찾아 복사 편집하는 데 시간이 오래 걸린다.
- 직무별 표현 수준이 들쭉날쭉하다.

## 기대 효과
- JD 초안 작성 리드타임 단축
- 표현 품질 표준화

## 범위
- 직무명 입력 기반 JD 초안 생성
- 기존 JD 샘플 참조

## 참고 자료
- 기존 JD 모음
- 직무기술서 작성 가이드
```
