# Agent Task Page Template

이 템플릿은 실제 Agent 과제 페이지에 그대로 적용한다.

## 1. 페이지 위치

아래 둘 중 하나에 만든다.

`W/G` 과제:

```text
W/G > 도메인 > 과제명
```

사업부 과제:

```text
VD/DA/MX/... > 과제명
```

예:

```text
VD > JD 초안 생성 에이전트
```

```text
W/G > 채용 > JD 초안 생성 에이전트
```

## 2. 페이지 제목

페이지 제목은 곧 `과제명`이다.

예:

```text
JD 초안 생성 에이전트
```

권장:

- `[채용]` 같은 접두사는 붙이지 않는다.
- 제목 자체가 매트릭스 셀에 표시된다.

## 3. 페이지 레이블

예시 1: `VD`가 만든 채용 과제

```text
agent-task
owner-vd
dm-talent-acquisition
cell-vd-talent-acquisition
```

예시 2: `W/G > 제도` 과제

```text
agent-task
owner-wg
dm-policy
cell-wg-policy
```

## 4. 페이지 속성 표

`페이지 속성` 매크로를 먼저 삽입한 다음, 그 안에 아래 2열 표를 만든다.

| 항목 | 값 |
| --- | --- |
| 개발주체 | VD |
| 도메인 | 채용 |
| 상태 | 진행중 |
| 담당 | 홍길동 |
| 우선순위 | High |
| 완료예정일 | 2026-04-30 |
| 목적 | 채용 담당자의 JD 초안 작성 시간을 줄인다 |
| 관련 링크 | Jira-123 / 문서 링크 / 데모 링크 |

중요:

- 왼쪽 열에는 제목 스타일을 적용한다.
- 컬럼명은 페이지마다 동일하게 유지한다.
